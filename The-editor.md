Lua RTOS has an integrated editor that allows you to edit files.

|Key| Function |
|---|----------|
| Ctrl+S | Save file |
| Ctrl+Q | Quit |
| up | Move one line up |
| down | Move one line down |
| left | Move one character left |
| right | Move one character right |
| right | Move one character right |
| page up | Move one page up |
| page down | Move one page down |
